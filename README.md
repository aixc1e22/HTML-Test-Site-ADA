# ADA — Adaptive AI Agent for Assessment Design
## University of Southampton · Centre for Higher Education Practice

This repository contains the ADA knowledge base website, designed to be hosted on GitHub Pages as a public-facing reference for the ADA AI agent.

### What this site contains

- **index.html** — Homepage with ALF overview and theoretical foundations
- **persona.html** — ADA's persona, quality standards, facilitation principles, and conversation flow
- **framework.html** — The Collaborative Assessment Design Decision Making Framework (4 phases)
- **alf.html** — Full detail of all 15 ALF criteria (F1–F6, N1–N4, G1–G5)
- **simulation.html** — Simulation mode instructions and scenario starters
- **tilt.html** — The TILT (Transparency in Learning & Teaching) methodology
- **style.css** — Shared stylesheet

### Deploying to GitHub Pages

1. Create a new GitHub repository (can be public)
2. Upload all files in this folder to the repository root
3. Go to **Settings → Pages**
4. Under **Source**, select **Deploy from a branch**
5. Select **main** branch and **/ (root)** folder
6. Click **Save**
7. Your site will be live at `https://[your-username].github.io/[repo-name]/`

### Using with Copilot

Once deployed, paste the URL of each page into Copilot's knowledge base or reference it in your system prompt. Since the site is public and crawlable by Bing, Copilot should be able to reference the content directly.

Suggested Copilot system prompt opening:
```
You are ADA (Adaptive AI Agent for Assessment Design). Your full knowledge base is available at [YOUR GITHUB PAGES URL]. Reference this site for all ALF criteria, facilitation principles, framework phases, TILT methodology, and simulation mode instructions.
```
